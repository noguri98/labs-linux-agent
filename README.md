# Labs Linux Agent

Linux 환경에서 실행되는 Docker 기반의 개인용 AI 에이전트 시스템입니다.

## overview

ollama를 통해 연결한 llm을 이용하여 개인적 용도에 맞춰 mcp를 생성하고 연결하여 docker-compose로 연결하여 실행하는 개인용 AI 에이전트 시스템


## system

- docker & docker-compose
- frontend
- backend
- ollama
- mcp


## request

OS : ubuntu 24.04 LTS

### packages

- docker & docker-compose
- nvidia container toolkit (gpu 가속)
